Threat Hunt · Investigation

Hunt 11: Just Another Day — Compromised Account

A billing analyst account (j.morris) was flagged for suspicious activity. The investigation revealed a living-off-the-land (LOTL) intrusion — RDP access from external IPs, reconnaissance, lateral movement, and HR data collection using only legitimate credentials and built-in Windows utilities. No malware or exploits involved.

Log N Pacific Cyber Range Microsoft Sentinel Defender Advanced Hunting (KQL) Mar 8–18, 2026

Attack Chain

  • Initial Access — Multiple RemoteInteractive (RDP) logons from external IPs (e.g. 193.36.225.245), inconsistent with the user's normal behavior.
  • Reconnaissance — Native commands to map the environment: whoami, hostname, net use, net view /domain, arp, nslookup.
  • Lateral Movement — Pivoted over RDP to an IT workstation and the file server (primary objective).
  • File Tampering — Edited approved billing records, planted a file renamed to match the invoice convention, and altered the billing audit trail to blend in.
  • Collection — Broad access to sensitive HR data: payroll, employee awards, onboarding, and payroll review files.

Root Cause

Compromised user credentials. The attacker authenticated successfully with the billing analyst account and performed all activity with legitimate Windows utilities (cmd, net, arp, nslookup, mstsc, whoami) — no malware or exploitation identified.

Detection Opportunities

  • Monitor RDP logons originating from external / non-corporate IPs.
  • Baseline file-share access by department and alert on out-of-role access.
  • Detect files created and immediately renamed to match naming conventions in sensitive directories.
  • Alert when audit logs or financial approval records change outside normal workflows.
  • Correlate RDP with native admin command execution to catch hands-on-keyboard activity.
